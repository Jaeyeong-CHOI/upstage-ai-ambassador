📧 이메일 설정 가이드

1단계: Google 2단계 인증 활성화

  1. myaccount.google.com 접속
  2. 왼쪽 메뉴에서 보안 선택
  3. Google에 로그인하는 방법 → 2단계 인증 클릭
  4. 안내에 따라 활성화 (이미 켜져 있으면 다음 단계로)

📖 Google 공식 가이드 · 2단계 인증이 없으면 앱 비밀번호를 만들 수 없습니다.

2단계: 앱 비밀번호 생성

  1. myaccount.google.com/apppasswords 접속
  2. 앱 이름 입력 (예: n8n credential)
  3. 만들기 클릭
  4. 화면에 표시되는 16자리 비밀번호를 복사 (다시 볼 수 없음!)

📖 앱 비밀번호 공식 가이드 · 일반 Gmail 비밀번호가 아닌 별도의 16자리 앱 전용 비밀번호입니다.

3단계: n8n에서 크레덴셜 만들기

워크플로우의 Send Email 노드를 클릭하면 크레덴셜을 설정할 수 있습니다.

  1. Send Email 노드 더블클릭
  2. 상단의 Credential to connect with → Create New Credential 클릭
  3. 아래 값을 입력:
UserGmail 주소 (예: me@gmail.com)
Password앱 비밀번호 16자리 (2단계에서 복사한 것)
Hostsmtp.gmail.com
Port465 (기본값, SSL용)
SSL/TLS✅ On
  1. Save 클릭 → 크레덴셜 저장 완료

📖 n8n 공식 Gmail SMTP 설정 가이드

4단계: 이메일 주소 설정 및 테스트

  1. Send Email 노드에서 From Email과 To Email을 본인 주소로 수정
  2. 또는 n8n Settings → Variables에서 환경변수로 관리:
    SMTP_FROM = 보내는 주소
    DIGEST_TO = 받는 주소
  3. 워크플로우에서 Test workflow 버튼 클릭 (Manual Trigger 실행)
  4. 받은 편지함에서 이메일 확인! 🎉
